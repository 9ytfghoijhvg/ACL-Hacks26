# ACL-Hacks26 - DissCourse
Aarav, Rishan, and Anish's ACL Hacks Project: DissCourse. 
DissCourse is a website where you can talk to clones of famous people and make them debate on any topic you want.
This can be used for educational purposes to learn about the perpectives of people that might have not been alive at the same time as the other person
This can also be used for fun purposes to see how the AI's would react to various silly topics
This project was made with ReactJS as the frontend and Python as the backend. FastAPI was used for frontend-backend communication while ollama was used for the AI's inside of the program. Vite was used to build the website.
How To Run The Program:
 In order to run, you need to install NodeJS, Npm, and Python. In a terminal, first run "cd DisCourse" to go to the directory and then run "python3 src/debate.py"
 After that, open up a new terminal window (don't close the python one), run "cd DisCourse", and then run npm run dev. Open up the localhost that it gives you, and have fun chatting!
